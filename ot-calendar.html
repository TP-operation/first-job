20 ก.ค. 2569

กดเลือกชั่วโมงการทำงาน (ระบบจะบันทึกทันที)

ซิงค์ข้อมูลข้ามเครื่อง

ตั้งค่าครั้งเดียว ใช้ได้ทุกเครื่อง

  1. ไปที่ console.firebase.google.com แล้วสร้างโปรเจกต์ใหม่ (ฟรี)
  2. กด "เพิ่มแอป" เลือกไอคอนเว็บ </> ตั้งชื่ออะไรก็ได้ แล้วคัดลอกโค้ด firebaseConfig ทั้งก้อน
  3. เมนูซ้าย Build → Firestore Database → กด "Create database" (โหมดไหนก็ได้)
  4. แท็บ Rules ของ Firestore วางกฎนี้แทนของเดิม แล้วกด Publish: 
    rules_version='2';
service cloud.firestore {
  match /databases/{db}/documents {
    match /otSync/{pin} {
      allow read, write: if request.auth != null;
    }
  }
}
  5. เมนูซ้าย Build → Authentication → Sign-in method → เปิดใช้ "Anonymous"
  6. วางโค้ด firebaseConfig ที่คัดลอกมาลงช่องด้านล่าง ตั้ง/กรอกรหัสซิงค์ 6 หลัก แล้วกดเชื่อมต่อ (เครื่องแรกกด "สุ่มรหัสใหม่" แล้วจดรหัสไปกรอกที่เครื่องอื่นให้ตรงกัน)

เครื่องอื่นๆ ต้องกรอกรหัสเดียวกันนี้เพื่อให้ข้อมูลเจอกัน
