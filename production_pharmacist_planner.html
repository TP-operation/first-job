ตั้งค่าข้อมูล Google Cloud API

การซิงก์ข้อมูลโดยตรงไปยัง Google Drive ส่วนตัวของคุณ จำเป็นต้องใช้คีย์ที่ออกโดยบัญชี Google Developer ของคุณเอง (ทำตามคู่มือสั้นๆ ได้ฟรีโดยไม่เสียค่าบริการ)

🛠️ วิธีตั้งค่าสั้นๆ:

  1. เข้าสู่ Google Cloud Console สร้างโปรเจกต์ใหม่
  2. เปิดใช้งาน **Google Drive API** ค้นหาและกด Enable
  3. ตั้งค่าสิทธิ์ที่ **OAuth Consent Screen** (เลือกประเภท External / Testing) และเพิ่ม Email ของคุณในฐานะ Test Users
  4. สร้างสิทธิ์เข้าใช้งานที่ **Credentials** -> สร้าง OAuth Client ID (เลือก Web Application, ในหัวข้อ *Authorized JavaScript Origins* ใส่ http://localhost หรือ URL ปัจจุบันที่คุณเปิดหน้าเว็บนี้)
  5. สร้าง API Key เพิ่มเติมในหน้านั้นและคัดลอกข้อมูลทั้ง 2 มากรอก
